# 🛍️ Myntra Clone Website (Frontend Project)

A clean and responsive **Myntra Clone UI** built using **HTML, CSS, and JavaScript**.  
This project replicates the basic layout of Myntra including the navbar, search bar, category scrolling cards, and login pages.

---

## 🌟 Key Highlights

✅ Myntra-like UI design  
✅ Smooth horizontal scrolling categories  
✅ Banner + category sections  
✅ Login page with validation  
✅ Show/Hide password feature  
✅ Password confirmation check (match / not match)  
✅ Beginner-friendly frontend project  

---

## 🛠️ Tech Stack Used

- **HTML5**
- **CSS3**
- **JavaScript**
- **Google Material Symbols Icons**

## 📌 Pages Included

### 🏠 1. Homepage (`index.html`)
- Header with Myntra logo and navbar
- Search bar
- Profile / Wishlist / Bag icons
- Banner section
- Category cards in scroll view
- Footer section

### 🔐 2. Login Page (`login.html`)
- Username + password login form
- Show password option
- Validates demo credentials

### 🔁 3. Password Confirmation Page (`password.html`)
- Password and confirm password match check
- Shows live message:
  - ✅ Password confirmed
  - ❌ Password does not match
